# 摸鱼盯盘助手

上班摸鱼神器：一个极简的、隐蔽的 Python 桌面盯盘小工具。

## 功能特点

*   **极简悬浮窗**：置顶显示，背景半透明，融入桌面。
*   **双模式显示**：
    *   **百分比模式**：极简文字，适合隐蔽摸鱼。
    *   **柱状图模式**：可视化红绿条，直观感受涨跌幅度（支持动态量程与圆角UI）。
*   **实时更新**：对接腾讯/新浪财经接口，实时刷新。
*   **智能记忆**：自动保存自选股列表及显示模式偏好。
*   **老板键**：双击隐藏/显示。
*   **配置简单**：右键菜单添加/删除股票。
*   **到价提醒**：盈亏转正或突破整数关口时抖动提醒。

## 使用方法

### 方式一：直接运行 (Windows)
双击 `stock_monitor.exe` 即可运行。

### 方式二：源码运行
需要安装 Python 3 和相关依赖：
```bash
pip install requests
python stock_monitor.py
```

## 开发说明
本项目使用 `tkinter` 构建 GUI，`requests` 获取数据。
